👋 Bienvenido al Sistema de Gestión de Bienestar

Gracias por utilizar nuestra aplicación móvil. Este sistema ha sido diseñado para facilitar la gestión de servicios de bienestar, permitiéndote administrar clientes, servicios, citas y facturas desde tu dispositivo móvil de manera sencilla e intuitiva.

Gestionar Clientes

⚠️ Acceso restringido: Esta función solo está disponible para usuarios con rol de Administrador.

Los administradores pueden gestionar la información de los clientes: agregar nuevos clientes, editar información existente y eliminar registros.

Pantalla de gestión de clientes

Agregar un nuevo cliente

Paso 1: Abrir formulario

Toca el botón flotante + en la esquina inferior derecha

Paso 2: Completar datos

Se abrirá un formulario donde debes ingresar:

  • Nombre completo: Nombre y apellido del cliente
  • Email: Correo electrónico válido
  • Teléfono: Número de contacto

Paso 3: Guardar

Presiona el botón Guardar. Si los datos son correctos, el cliente se agregará a la lista.

Editar un cliente existente

  1. Busca el cliente en la lista
  2. Toca el botón ✏️ (editar)
  3. Modifica los datos necesarios en el formulario
  4. Presiona Guardar para aplicar los cambios

Eliminar un cliente

  1. Busca el cliente en la lista
  2. Toca el botón 🗑️ (eliminar)
  3. Confirma la acción en el cuadro de diálogo
⚠️ Cuidado: Eliminar un cliente es una acción permanente. Asegúrate de que realmente deseas eliminar el registro antes de confirmar.
💡 Validaciones:
  • El nombre no puede estar vacío
  • El email debe tener formato válido (ejemplo@dominio.com)
  • El teléfono no puede estar vacío

ℹ️ Nota: Actualmente esta función está en desarrollo. Próximamente estará disponible con todas las funcionalidades.

¿Cómo saber mi rol?

Tu rol se muestra en el menú principal, justo debajo del mensaje de bienvenida. Si necesitas permisos adicionales, contacta al administrador del sistema.

💡 Para administradores: Los roles se asignan desde la base de datos. Consulta el manual técnico para instrucciones sobre cómo crear usuarios con roles específicos.

❓ Preguntas Frecuentes (FAQ)

1. ¿Cómo cierro mi sesión?

Toca el botón Salir en la barra superior del menú principal. Esto cerrará tu sesión de forma segura y te regresará a la pantalla de login.

2. ¿Qué hago si olvidé mi contraseña?

Actualmente no existe función de recuperación automática. Contacta al administrador del sistema para que restablezca tu contraseña.

3. ¿Por qué no veo el botón "Ver Clientes"?

Esta función solo está disponible para usuarios con rol de Administrador. Si eres cliente o recepcionista, no tendrás acceso a esta opción.

4. ¿Puedo usar la aplicación sin conexión a internet?

No, actualmente la aplicación requiere conexión a internet para funcionar, ya que se comunica con el servidor en tiempo real. En futuras versiones se implementará soporte offline.

5. ¿Los datos están seguros?

Sí, el sistema implementa:

  • Autenticación segura con sesiones
  • Cifrado de comunicaciones (HTTPS en producción)
  • Control de acceso basado en roles
  • Validación de datos en servidor y cliente

6. ¿Cómo reporto un problema o error?

Si encuentras un error o problema:

  1. Anota qué estabas haciendo cuando ocurrió el error
  2. Toma una captura de pantalla si es posible
  3. Contacta al administrador o soporte técnico
  4. Proporciona tu nombre de usuario y descripción del problema

7. ¿La aplicación se actualiza automáticamente?

Las actualizaciones del backend se realizan en el servidor sin que lo notes. Para actualizaciones de la aplicación móvil, deberás descargar la nueva versión cuando esté disponible.

8. ¿Qué navegadores son compatibles?

Esta es una aplicación móvil nativa para Android. No requiere navegador web. Funciona en dispositivos con Android 7.0 (API 24) o superior.

9. ¿Puedo acceder desde iOS (iPhone/iPad)?

Actualmente solo existe versión para Android. Una versión para iOS podría desarrollarse en el futuro si hay demanda.
